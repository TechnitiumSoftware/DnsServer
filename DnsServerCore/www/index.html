Forgot Password?

                

                

                    
Follow these steps to reset 'admin' password:

                    
    
                        
  1. Find the DNS Server config folder and locate the dns.config file. The config folder will be found where the DNS Server is installed.
    2. 
                        
  2. Rename the dns.config file as reset.config
    3. 
                        
  3. Restart the DNS Server to complete the password reset process.
    4. 
                        
  4. Just refresh this web page in the web browser to auto login with default credentials and quickly change the password.
    5. 
                    

                    
Note: To reset 'admin' password, you will need file system access on the server running this DNS Server.
